📚 API Documentation - Santuarios Media Upload

POST /api/upload

Descripción: Sube una imagen o video a Supabase Storage

Content-Type: multipart/form-data

Body:

  • file - Archivo a subir (imagen o video)

Respuesta exitosa (200):

{
  "success": true,
  "data": {
    "fileName": "images/1234567890-abc123.jpg",
    "fileType": "image",
    "mimeType": "image/jpeg",
    "size": 102400,
    "publicUrl": "https://xxx.supabase.co/storage/v1/object/public/...",
    "path": "images/1234567890-abc123.jpg"
  },
  "message": "Archivo subido exitosamente"
}

DELETE /api/delete

Descripción: Elimina un archivo de Supabase Storage

Query Parameters:

  • path - Ruta del archivo a eliminar (ej: images/1234567890-abc123.jpg)

Ejemplo: /api/delete?path=images/1234567890-abc123.jpg

Respuesta exitosa (200):

{
  "success": true,
  "message": "Archivo eliminado exitosamente"
}

GET /api/list

Descripción: Lista archivos en Supabase Storage

Query Parameters (opcionales):

  • folder - Carpeta a listar (default: raíz)
  • limit - Cantidad máxima de resultados (default: 100)
  • offset - Offset para paginación (default: 0)

Ejemplo: /api/list?folder=images&limit=10

Respuesta exitosa (200):

{
  "success": true,
  "data": [
    {
      "name": "1234567890-abc123.jpg",
      "path": "images/1234567890-abc123.jpg",
      "size": 102400,
      "createdAt": "2026-03-06T10:00:00.000Z",
      "publicUrl": "https://xxx.supabase.co/storage/v1/object/public/..."
    }
  ],
  "total": 1,
  "folder": "images"
}
